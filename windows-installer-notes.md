# Windows Installer Notes

## If Windows SmartScreen appears:

1. Click "More info"
2. Click "Run anyway"

The app is safe - SmartScreen appears because the app is not signed with an expensive code signing certificate.

## Installation Location

By default, Redis Tics installs to:
- `%LOCALAPPDATA%\Programs\Redis Tics`

## Uninstallation

Use Windows Settings > Apps > Redis Tics > Uninstall
