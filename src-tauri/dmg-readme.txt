Redis Tics - Installation Instructions
==========================================

1. Drag the Redis Tics app to the Applications folder
2. If you see a "damaged" or security warning, run this command in Terminal:

   xattr -cr /Applications/Redis\ Tics.app

3. Launch Redis Tics from your Applications folder

For support and documentation:
- Website: https://redistics.com
- GitHub: https://github.com/pritamsso/redis-tics

Thank you for using Redis Tics!
